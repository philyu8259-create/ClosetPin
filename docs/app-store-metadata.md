## English Listing

Name:

ClosetPin

Subtitle:

AI outfit planner

Promotional text:

Build a visual closet, try a demo wardrobe first, and let AI help narrow tomorrow's outfit without overcomplicating your routine.

Description:

ClosetPin helps you turn your wardrobe into a calmer daily decision.

Add clothing from photos, organize pieces in a visual closet, and get outfit suggestions for workdays, meetings, banquets, weekends, and tomorrow's weather. If you are new, start with the demo closet first to understand how recommendations, saved looks, and outfit feedback work before adding your own clothes.

Key features:

- Visual wardrobe grid for tops, bottoms, shoes, outerwear, and bags
- AI-assisted photo recognition with review-before-save controls
- Outfit recommendations for daily work, meetings, banquets, and weekends
- Tomorrow weather context for outfit preparation
- Saved looks and feedback to improve future suggestions
- Local-first wardrobe data, with cloud recognition only when enabled
- Pro subscription for advanced AI outfit assistance

ClosetPin is designed to keep the flow simple: add clothes, review suggestions, choose a scenario, and decide what to wear. It is not a social feed and not a complicated inventory system. It is a practical wardrobe assistant for people who want better outfits with less morning friction.

Keywords:

closet,wardrobe,outfit,style,AI stylist,clothes,fashion,planner,weather,packing

What's New:

Added a user-facing demo closet so new users can explore recommendations, the closet grid, and saved looks before adding their own clothes.
